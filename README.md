# Calculator
Intive first project

## Operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Additional features:
- operations on numbers with commas
- Equal sign (=) - returns the result of the last operation
- "C" - deletes the entire entered expression (all memory)
- "CE" - allows the digit to be entered again
